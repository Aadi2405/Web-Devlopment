HTML Basics

HTML is the structure of web pages.

It uses tags like <h1>, <p>, and <div> to define elements.

<h1>Hello World</h1> <p>This is a paragraph.</p>

HTML is not case sensitive but follows conventions.

  • Headings
  • Paragraphs
  • Links
CSS Styling

CSS makes your website look better.

Use selectors to apply styles to HTML elements.

p { color: blue; }

Stylesheets can be internal or external.

.container { margin: 20px; }
  • Colors
  • Fonts
  • Spacing
JavaScript Intro

JavaScript adds interactivity to websites.

You can respond to user actions like clicks and input.

document.getElementById("btn").addEventListener("click", myFunction);

JavaScript can also change the HTML content dynamically.

element.innerHTML = "Hello!";
  • Variables
  • Functions
  • DOM Manipulation
Responsive Design

Responsive design helps your site work on all devices.

Use CSS media queries to adjust layout based on screen size.

@media (max-width: 600px) { body { font-size: 14px; } }

It ensures good user experience everywhere.

.flex-container { display: flex; flex-wrap: wrap; }
  • Media Queries
  • Flexible Layouts
  • Mobile-first Design
